# 🎵 DesidiaMX API

API para gestión de álbumes musicales desarrollada con spring.

---

## 📥 Clonar Repositorio

```bash
git clone https://github.com/EderAJ30/Desidia_MX.git
cd DesidiaMX
```

---

## 🐳 Ejecutar con Docker

### 🔧 Construir y levantar los contenedores

```bash
docker-compose build
docker-compose up
```

### ♻️ Reiniciar entorno desde cero

```bash
docker-compose down -v
docker-compose up -d
```

---

## 🌐 URL de la API

```
http://localhost:8081/api/albumes/
```
